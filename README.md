
# Python-Loops-Control-Statements-and-Functions


## Introduction

This project demonstrates the use of Python programming concepts including loops, control statements, and functions. The assignment consists of three tasks:

Number Guessing Game using a While Loop
Multiplication Table Generator using a For Loop
BMI Calculator using Functions

The programs accept user input, perform calculations, and display results based on logical conditions.
## Topics Covered

Python Basics

User Input and Output

While Loop

For Loop

Range Function

Control Statements (break, continue, else)
Functions

Parameters and Return Values

Conditional Statements (if, elif, else)

Random Number Generation

Mathematical Calculations
## Key Insights

Learned how to use a while loop to control program execution based on a condition.

Understood the purpose of break and continue statements in loop control.

Used a for loop with the range() function to generate repetitive outputs efficiently.

Created reusable code using functions.

Implemented real-world applications such as a number guessing game and BMI calculator.

Improved problem-solving and logical thinking skills through practical programming tasks.

This assignment provided hands-on experience with Python loops, control statements, and functions.
